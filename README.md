## 🛠️ Tech Stack
- *Framework:* Next.js 14 (App Router)
- *Database:* Supabase (PostgreSQL)
- *Scraping:* Axios & Cheerio
- *Styling:* Tailwind CSS

## ⚡ Features
- *URL Scraping:* Instantly extract text data from any web link.
- *Auto-Sync:* Saves scraped content directly to Supabase.
- *Professional UI:* Dark-themed, modern interface.

## ⚙️ Setup
1. Clone the repo: git clone https://github.com/abubakarsabir924/SiteMind-AI.git
2. Install: npm install
3. Add .env.local with your Supabase keys.
4. Run: npm run dev
